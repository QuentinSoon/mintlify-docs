## Se connecter

Sur cette page il est possible de se connecter avec ses logins ou alors de changer son mot de passe si besoin.

![loginPage](/public/assets/doc-screen/doc-loginpage.png)

### La page après la connexion

<img width="30" noZoom src="/public/assets/doc-screen/RemarksIcon.png" /> Lorsqu’un
seul dataset existe, il se charge automatiquement.

<img width="30" noZoom src="/public/assets/doc-screen/RemarksIcon.png" /> Lorsque
plusieurs datasets existent, une liste est proposée afin de choisir le dataset de
travail.

![loginDataset](/public/assets/doc-screen/doc-logindataset.png)

Si la ligne organisation n'est pas disponible, c'est que le paramètre lié n'est pas actif. Pour pouvoir l'activer ou le désactiver, voir avec un administrateur.

### Numéro de version et documentation

Le menu « Paramètres » en haut à droite de l’écran vous permet d’accéder :

• Au numéro de version

• A la documentation technique et utilisateur

• Aux légendes utilisées dans l’application

• Aux couleurs utilisées pour les statuts

• A la synchronisation

![loginParameters](/public/assets/doc-screen/doc-loginparameters.png)

### Page Post-connexion

Durant toute la session, le menu de l’utilisateur est constamment accessible via cette icône de
connexion en haut et à droite de l’écran :

![UserSideBarOption](/public/assets/doc-screen/doc-UserSide.png)

Ce menu permet de :

• Se déconnecter

• Changer de Dataset

• Rafraîchir l’affichage du dataset actif

• Visualiser/Effacer les messages de l’application

Les différents onglets de l’application 7Opteam Calendar sont sur la partie haute de l’écran :

![MenuBar](/public/assets/doc-screen/doc-MenuBar.png)
